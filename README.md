
# NYC Taxi Trip Data Analysis (Python)

## Project Overview

This project performs exploratory data analysis on New York City taxi trip data using Python.
The goal is to understand travel patterns, trip duration behavior, and taxi usage trends.

## Dataset

The dataset contains over **1.4 million taxi trips** including pickup and dropoff times, passenger counts, and trip duration.

Dataset Source: Kaggle - NYC Taxi Trip Duration

## Analysis Performed

The following analyses were conducted:

* Data exploration and summary statistics
* Feature engineering (hour and weekday extraction)
* Taxi demand by hour of day
* Taxi demand by day of week
* Trip duration distribution
* Passenger count analysis
* Trip duration vs passenger count
* Correlation analysis

## Key Insights

* Taxi demand peaks during morning and evening commuting hours.
* Most taxi trips last between approximately 5–20 minutes.
* The majority of trips involve a single passenger.
* Trip duration does not significantly change with passenger count.
